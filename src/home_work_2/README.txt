Пакет loops

Задание 1.1.1	- Класс Factorial_1_1_1. Перемножение всех целых чисел от 1 до заданного значения.
Задание 1.1.2*	- Класс Factorial_1_1_2. Перемножение всех целых чисел от 1 до заданного значения (рекурсия).
Задание 1.2     - Класс NumberToDigits_1_2. Перемножение всех чисел, входящих в введенное число + проверки ввода.
Задание 1.3     - Класс Exponentation_1_3. Возведение первого введенного числа в степень второго введенного числа
                    + проверки ввода.
Задание 1.4*    - Класс Exponentation_1_4. Умножение числа long a = 1 на введенное значение до переполнения.
                    Вывод значения до переполнения и после.
Задание 1.5     - Класс MultTable_1_5. 	Создание таблицы умножения в консоли по шаблону.
Задание 2.1     - Класс ArraysUtils. Методы arrayFromConsole() - ввод с консоли, arrayRandom() - ввод
                    произвольных значений.
Задание 2.2     - Класс ArraysIteration. Ввод массива методом ArraysUtils.arrayFromConsole().
                    Вывод на экран каждого элемента, каждого второго элемента, каждого элемента в обратном порядке.
Задание 2.4     - Класс Arrays_2_4. Главный класс вызывает методы, реализующие все подпункты ДЗ (см. описание методов).
